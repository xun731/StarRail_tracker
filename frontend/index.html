      📖 取得抽卡網址(目前只適用於電腦)

      星穹鐵道沒有內建「複製抽卡 URL」按鈕，HoYoLAB App 也僅顯示已擁有角色，無法看抽卡紀錄。 因此唯一可行的做法是從遊戲本機快取讀取（遊戲打開抽卡頁時自己會寫入快取）。 以下兩種方法擇一即可。

      ⚡ 方法 A：使用 PowerShell 腳本（推薦，便利）

      ⚠ 風險聲明： 腳本會讀取星穹鐵道本機快取檔（data_2）並從中 regex 抽出 URL， 只讀不寫、不注入遊戲、不傳送資料到外部。 技術上違反 miHoYo TOS，但 Paimon.moe / Fribbel / Snap.Hutao / Starward 等社群工具都採同手法， 多年無大規模封號案例。風險不為零，使用即接受風險。

      取得腳本（三選一）：

      📥 下載腳本 🐙 GitHub 原始碼

      執行步驟：

      1. 啟動星穹鐵道，進入 躍遷 → 紀錄 頁面（讓遊戲寫入快取）
      2. 找到下載的 get_authkey.ps1，右鍵 → 「使用 PowerShell 執行」
      3. 若被 Windows 阻擋（紅字），改用 PowerShell 開啟並貼：
        powershell -ExecutionPolicy Bypass -File .\get_authkey.ps1
      4. URL 會自動進剪貼簿，回此頁 Ctrl+V 貼到下方輸入框
      🔒 方法 B：完全手動（零腳本，最透明）

      適合不信任任何腳本、想完全看到自己在做什麼的使用者。流程就是腳本做的事，只是用手做：

      1. 啟動星穹鐵道，進入 躍遷 → 紀錄 頁面
      2. 用檔案總管打開：
        {遊戲安裝路徑}\StarRail_Data\webCaches\<最新版本>\Cache\Cache_Data\data_2
      3. 用 Notepad++、VSCode 或任何純文字編輯器開啟 data_2
      4. Ctrl+F 搜尋 getGachaLog
      5. 找到 https://... 開頭、含 authkey= 的長 URL（取最後一個就是最新的）
      6. 整段複製，貼到下方輸入框

      💡 沒有 Notepad++ 也可以用 PowerShell 一行查（這條只「讀」沒「執行」）：
      Get-Content "..\data_2" -Raw | Select-String "https://[^\s]+getGachaLog[^\s]+authkey=[^\s]+"

      authkey 約 24 小時失效，過期重複以上任一方法即可。

      🔗 貼上抽卡網址

      ⚙ 進階：自訂後端位址（一般用戶不用碰）

      預設值會自動依目前網域選擇：本機開發 → http://localhost:5000； 線上版 → 雲端後端位址。除非你自架後端，否則維持預設即可。

      📁 或：從 CSV / Excel 檔案匯入

      若你已有 CSV 或 Excel 形式的抽卡紀錄（從其他工具匯出、自製試算表…），可直接上傳。 支援格式：.csv / .xlsx / .xls

      📌 欄位對應

      支援值：是/否、true/false、歪/未歪、1/0、有值/空白

      Excel 多半是六個月以前的舊紀錄，建議「插入到最下方」。

      📋 預覽抽卡紀錄

      勾選要匯入的紀錄；取消勾選你已手動輸入過的。 匯入紀錄會以 📥 標記，與手動紀錄一起排序顯示。

      ⚠ 「歪」的判斷可能不完整： 系統用 banner_history.js 中的 UP 對照表判斷。 若該抽出時間沒收錄對應 banner，會保守地不標歪， 匯入後可至「統計分析」頁手動點 ✏️ 修正。

      尚未有任何紀錄，請先在「新增紀錄」頁面新增資料

      開啟後每筆紀錄旁邊會出現 ▲▼ ✏️ ✕ 按鈕，可調順序、改內容、刪除

      ❓ 常見問題

      「歪」是什麼意思？
      在角色 / 光錐活動池中，五星有 50%（角色）或 75%（光錐）機率出目標 UP 角色 / 光錐。若出了非 UP 的常駐五星，稱為「歪」，並觸發大保底：下一個五星必定是 UP 目標。
      軟保底是什麼？
      從特定抽數開始（角色池第 74 抽、光錐池第 65 抽），每一抽的五星機率會大幅提升，直到硬保底（100% 出五星）為止。
      抽卡紀錄資料存在哪裡？
      登入 Google 帳號後資料儲存在雲端（Firebase），可跨裝置存取。未登入時資料僅存在瀏覽器本機，清除瀏覽器資料會遺失。
